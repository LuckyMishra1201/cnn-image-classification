# CNN Image Classification using AlexNet

## Overview
In this project, I built a deep learning model to classify natural scene images using a Convolutional Neural Network (CNN) based on the AlexNet architecture.

## What I did
- Used Intel Image Classification dataset with 6 classes (buildings, forest, glacier, mountain, sea, street)
- Preprocessed image data (resizing, normalization, batching)
- Implemented AlexNet-based CNN architecture
- Trained the model using Adam optimizer and evaluated performance
- Applied techniques like dropout and early stopping to prevent overfitting
- Analyzed results using accuracy, precision, recall, and confusion matrix

## Results
- Achieved ~84–86% accuracy on test dataset
- Model performed well across all classes with minor confusion in similar categories

## Tech Stack
- Python  
- TensorFlow / Keras  

## Learnings
- Understanding CNN architecture and feature extraction  
- Importance of preprocessing in image data  
- Impact of hyperparameters on model performance
